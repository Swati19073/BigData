Help


Users can follow the given instructions while predicting thyroid disease:

  1. T3: T3 or triiodothyronine is a type of hormone produced by thyroid which is mainly responsible for stabilizing blood pressure, body temperature and regulating heart rate. Users must enter the value of T3 between 0.0005-0.18.
  2. TT4: TT4 also known as levothyroxine is another type of hormones produced by thyroid which along with T3 regulates body temperature, metabolism and heart rate. Users can enter the value of TT4 between 0.002-0.6.
  3. TSH: TSH stands for Thyroid Stimulating Hormone. Pituitary glands of the brain produce TSH which tells the thyroid to produce and release the thyroid hormones into your blood. Users can enter the value of TSH between 0.0-0.53.
  4. T4U: Its value ranges from 0.002 to 0.6.
  5. FTI: FTI stands for Free Thyroxine Index which is obtained by multiplying (Total T4) times (T3 Uptake) to obtain an index. In the presence of abnormalities in plasma protein binding, the FTI indicator is considered as a reliable indicator of thyroid status.

Prediction Approach


For the prediction of thyroid Decision Tree Classifier is used in which the model is trained using the above 5 features by keeping the depth of the tree as 3. After performing normalization, the Decision Tree Classifier was able to predict thyroid with an accuracy of 97.22%.


Output


The output obtained can be either 1, 2 or 3 representing normal, hyperthyroidism and hypothyroidism respectively.
